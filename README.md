## Setup

```
$ git clone https://github.com/jsbros/vimd
$ cd vimd
$ yarn install
```

## Usage

### Run Development Server

```
$ yarn start
```

### Run Tests (Powered by Jest)

```
$ yarn test
```

### Lint Code

```
$ yarn lint
```

### Generate Documentation

```
$ yarn docs
```

### Build Production Bundle

```
$ yarn build
```
